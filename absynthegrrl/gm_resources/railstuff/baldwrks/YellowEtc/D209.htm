DM&IR #209

Duluth,Missabe, And Iron Range, Class "M1s", 2-8-8-2 #209

Taken Procter, Minn. May 28, 1952.

Originally #209 Duluth, Missabe and Northern Ry. Class M. Built by Baldwin, 1910, C/N 43531.

D.M.& I.R. retained #209.

Specifications as built.

Cylinders = 26" & 40" x 32", Driver Dia. = 57", Wt. = 470,100 lb. Tr. Force = 91,000 lb.

Specifications as Simpled, class M1, 1937.

Rebuilt as Class M1s, 1953.

Cylinders = 26" x 32", Driver Dia. = 57", Wt. = 494,450 lb. Tr. Force = 110,000 lb.

Sold for scrap,1958

Test Sites

There exist several test sites for arcanes desiring to pass between the different order groupings. These sites are administered by several of the highest-ranking arcanes from the major schools and regions. For additional ideas about these sites see the file Magic User Test Site.

Kayara was given free hand in the design. It needs to show her total whimsy. Remember - there are map rooms near the test chambers.

In recent times, Kayara and Stormy have contracted Irena and arranged to use a Hut pocket area as the testing grounds. There is an access cave that links to the Mountain Side to allow access for the Hut crew. This entrance is turned off when actual testing is in progress to prevent the applicant from gaining access to the Hut. The testing grounds exist in the clones, but not the functionality, making it just another place in the Hut.

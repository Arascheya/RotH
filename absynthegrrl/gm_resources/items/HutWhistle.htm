Hut Whistle

The whistle stick part is made of teak and the skull is carved from old bone. The eye sockets are inlaid with obsidian and the nose hole is the mouth hole for the whistle part.

0 0 0 0 opens communications link with hut and alerts Mistress - it sets navsys in case controller desires to jump to whistle location.

0 0 0 1 summons cauldron if it is within 5 zeems. If the cauldron is within the hut, it will cause it to come out and expand to transport size.

0 0 1 0 turns cauldron 6th Order screen on and prompts for pass characteristics of screen.

0 0 1 1 turns 6th Order screen off

0 1 0 0 unassigned - can be set by Mistress for special function of hut or cauldron

0 1 0 1 fires one of cauldron's fireballs

0 1 1 0 causes cauldron to physically attack

0 1 1 1 stops physical attack mode

1 0 0 0 causes cauldron to move forward

1 0 0 1 increases movement speed by increments, one step for each sounding of the whistle

1 0 1 0 decreases movement speed by increments, one step for each sounding of the whistle

1 0 1 1 sends cauldron back to hut where it will automatically shrink and store itself on the shelf near the stove in the entry room.

1 1 0 0 turns cauldron left, amount of turn determined by length of whistle tone

1 1 0 1 turns cauldron right, amount of turn determined by length of whistle tone

1 1 1 0 moves cauldron higher into the air, height determined by length of whistle tone

1 1 1 1 moves cauldron lower into the air, height determined by length of whistle tone

This item is found in Margla's personal quarters
